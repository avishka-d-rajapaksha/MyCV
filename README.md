# My CV - Avishka D. Rajapaksha

A **professional multi-page CV/Resume website** built with pure HTML and CSS. Showcases technical skills, work experience, education, projects, and achievements in a modern, print-ready design.

## Key Features

- **Multi-Page Layout** — 3-page professional CV design
- **Sidebar with Contact Info** — Phone, email, LinkedIn, and location
- **Technical Skills Visualization** — Animated skill progress bars (Java, MySQL, MVC, OOP, React, Git)
- **Project Showcase** — Linked project cards with screenshots and tech stacks
- **Work Experience Timeline** — Professional history at Printcare Group
- **Education Section** — UoVT, SLIATE HNDIT, and A/L qualifications
- **Awards** — IEEE XtremeIgnite 1.0 (3rd Place)
- **Print-Ready** — Optimized for A4 printing

## Technologies & Tech Stack

| Technology | Purpose |
|---|---|
| HTML5 | Semantic page structure |
| CSS3 | Professional styling and layout |
| Responsive Design | Mobile and print compatibility |

## Setup and Installation

1. Clone the repository:
   ```
   git clone https://github.com/avishka-d-rajapaksha/MyCV.git
   ```
2. Open `index.html` in any modern web browser.
3. To print, use `Ctrl+P` and select A4 paper size.

## Usage

- View the CV in any web browser
- Print directly for physical copies
- Customize content by editing the HTML file
